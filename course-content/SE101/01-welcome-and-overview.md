## Welcome to SE101

This course is your entry point into software engineering at ADU. By the end you will write Python programs that solve real problems, understand how computers execute code, and think algorithmically.

### What You Will Learn

- Python syntax and core data types
- Control flow: conditions and loops
- Functions and modular code
- Working with files and data
- Algorithmic thinking and problem decomposition

### Tools You Need

Install Python 3.11+ from **python.org** and VS Code with the Python extension.

```bash
python --version   # should print Python 3.11+
pip --version      # package manager
```

### How This Course Works

Each lesson builds on the previous one. Complete the exercises at the end of each lesson before moving on. The final lesson is a mini-project that ties everything together.
