# Data Engineering — Detailed Course Notes

This document provides a structured lesson-by-lesson set of detailed notes for every registered course in this school. The lessons are sequenced from Lesson 1 to Lesson 6 for each course to support teaching, self-study, and LMS publishing.

## 1. DE101 — Introduction to Data Engineering

**Credits:** 3  
**Course focus:** What is Data Engineering and how does it differ from Data Science and Data Analytics? The Data Engineering Lifecycle: data generation, data storage, data ingestion, and data serving. Key skills and responsibilities of a data engineer in 2026. ## 4. DE104 — Networking Fundamentals & Distributed Systems Basics
